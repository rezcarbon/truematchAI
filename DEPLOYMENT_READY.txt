📊 SYSTEM ARCHITECTURE

Request Flow:
  Client Request
    ↓
  FastAPI Route (app/main.py)
    ↓
  DSAR Router (app/api/v1/dsar.py) — if /api/v1/dsar/*
    ↓
  Assessment Assessment Task Queue
    ↓
  Governance Engine
    ├─ Gate 1: Coherence Check
    ├─ Gate 2: Consistency Check
    ├─ Gate 3: Fidelity Assessment
    └─ Gate 4: Bias Detection
    ↓
  Claude API (with Circuit Breaker + GDPR Redaction)
    ↓
  Database Storage
    ├─ governance_logs (audit trail)
    ├─ dsar_requests (compliance)
    └─ assessments (with gates_passed_at)

Autonomous Background Tasks:
  • retention_daily_sweep (24h interval) — deletes old assessments
  • export_dsar_data (on-demand) — exports user data for download

  • handle_assessment_dlq (on-failure) — dead letter queue processing
